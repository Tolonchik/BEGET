Бегущая строка Бегущая строка – это анимационный эффект, при котором строка текста перемещается по странице один раз, несколько раз или постоянно. Для создания бегущей строки используется специальный тэг &lMARQUEE>
. Атрибуты этого тэга контролируют параметры перемещения текста. 1) BEHAVIOR – способ выполнения эффекта бегущей строки: a. Scroll –текст исчезает за краем страницы; b. Slide – после выполнения заданного числа циклов, текст остается у левого или правого поля страницы; c. Alternate – направление перемещения текста в строке меняется на противоположное, после выполнения каждого цикла. По завершению последнего цикла текст остается у края страницы. 2) BGCOLOR – цвет фона бегущей строки. 3) DIRECTION – направление перемещения текста: a. Left – влево; b. Right – вправо. 4) HEIGHT – высота бегущей строки. 5) HSPACE – отступ в пикселях текста бегущей строки от левого и правого полей. 6) LOOP – число показов текста в строке: a. ? – целочисленное значение, указывающее число повторов; Infinite – бесконечное повторение эффекта по циклу. 7) SCROLLAMOUNT – смещение текста в пикселях за один шаг. 8) SCROLLDELAY – временной промежуток между смещениями в миллисекундах, по умолчанию 60 мс. 9) VSPACE – отступ в пикселях по вертикали от текста до рамки бегущей строки. 10) TRUESPEED – минимальное значение смещения текста, по умолчанию 60 мс. 11) WIDTH – ширина бегущей строки в пикселях. 4. Графика на Web – страницах Добавление рисунка на web – страницу Рисунки, фотографии и декоративные элементы очень часто используются на web – страницах. В интернет используется только три графических формата: o JPEG o GIF o PNG Чтобы добавить графический объект в текст Web – страницы, используется тэг: &limg src=’имя файла’>
Атрибуту src присваивается путь к файлу на диске или URL – адрес файла рисунка в интернет. Рисунки, могут находиться в той же папке, в которой находится файл web – страницы, но лучше их размещать в отдельную папку и указывать путь доступа к ней. Пример: &limg src=”ris1.jpg”>
В данном примере рисунок должен находиться в той же папке, где и web – страница. Атрибуты тэга &limg>
Границы вокруг рисунка создаются с помощью атрибута BORDER тэга &limg>
По умолчанию обозреватель отображает рисунок без границ. Чтобы нарисовать границы вокруг рисунка нужно атрибуту BORDER указать значение толщины границы в пикселях. Пример: &limg src=”ris1.jpg” border=’10’>
С помощью атрибута ALT рисунок можно снабдить альтернативным текстом. Альтернативный текст отображается во время загрузки изображения или когда отображение рисунков отключено. Альтернативный текст также можно использовать как подсказку: при наведении курсора мыши на рисунок текст отобразится. Пример: &limg src=”ris1.jpg” border=’10’ alt=’рисунок на странице’>
С помощью атрибутов WIDTH и HEIGHT можно изменить размер изображения, увеличив или уменьшив его. При указании новой ширины и высоты необходимо помнить о пропорциональности исходных и новых размеров. 19 Пример: &limg src=”ris1.jpg” border=’10’ alt=’рисунок на странице’ width=100 height=100>

Связывание web – страниц при помощи гиперссылок Гиперссылки создаются на web – страницах для быстрого перехода внутри одной страницы (якорь), а также перехода с одной страницы на другую. Гиперссылки создаются следующим образом: &lA HREF=’адрес’>гиперссылка&l/A> При создании гиперссылки нужно установить два важных элемента, от которых зависит тип гиперссылки. 1) Адрес. В качестве адреса, назначаемого атрибуту HREF, можно использовать: a) Адрес URL на другой странице или путь на диске к другому документу HTML – наиболее часто используемый тип гиперссылок; b) Адрес URL на документ FTP – сервера; c) Ресурс в Интернет; d) Закладка на текущей Web – странице; e) Адрес электронной почты; f) Сценарий на языке JavaScript. 2) Гиперссылка. Между тэгами &lA>…&l/A> можно заключать: a) Любой текст, состоящий из одного или нескольких абзацев, предложение, слово, отдельные символы или буквы в слове (такие гиперссылки называются текстовыми); b) Графический объект, включая рисунки, фотографии и другую графику, добавляемую с помощью тэга &lIMG>, а также отдельные области внутри рисунка (такие гиперссылки называются графическими.) Пример: &lA HREF=’INDEX2.HTML’>самостоятельная работа &l/A> - это текстовая гиперссылка. &lA HREF=’INDEX2.HTML’>&limg src=’img1.gif’>&l/A> - это графическая гиперссылка. &lA HREF=’http://mail.ru’>Почта&l/A> - ссылка на ресурс в Интернет. Установка целевого окна открытия документа по гиперссылке По умолчанию гиперссылка открывает документ в текущем окне обозревателя. Но иногда необходимо открывать документы в новом окне, в другой рамке и т.д. для этого в ссылку добавляют атрибут TARGET со следующими параметрами. 1) ‘_blank’ – в новом окне 2) ‘_self’ – в текущей рамке; 22 3) ‘_parent’ – в рамке предыдущего уровня, если используется набор вложенных рамок. 4) ‘_top’ – в текущем окне обозревателя (по умолчанию) 5) ‘имя_рамки’ в рамке с указанным именем Модифицируйте предыдущий пример следующим образом: &lA HREF=’INDEX2.HTML’ target=’_blank’>самостоятельная работа &l/A> &lA HREF=’INDEX2.HTML’ target=’_top’>&limg src=’img1.gif’>&l/A> Проверьте, что произойдет при нажатии на ссылку. В документах HTML – таблицы часто используются для разметки структуры web –страницы на функциональные блоки: панель управления, панель заголовка, верхний и нижний колонтитулы, основной раздел и т.д. Создание таблиц в коде HTML Перед тем как вы приступите к вводу кода таблицы, нарисуйте макет на листе бумаги. Подсчитайте необходимое количество строк и столбцов, число ячеек в каждой строке и их размеры. Определите ячейки, которые необходимо объединить. После этого приступайте к написанию кода HTML. Таблица создается при помощи пары тэгов &lTABLE>…&l/TABLE>. У тэга &lTABLE> есть ряд атрибутов: a) WIDTH – ширина таблицы (либо в px, либо в %) b) HEIGHT – высота таблицы (либо в px, либо в %) c) BORDER – граница вокруг таблицы в px d) ALIGN- горизонтальное выравнивание таблицы e) CELLSPACING - устанавливает расстояние между ячейками (по умолчанию два пикселя); f) CELLPADDING - устанавливает отступ текста от края ячейки (по умолчанию один пиксель) Теперь нужно добавить в таблицу строки и ячейки. Для этого используются следующие тэги: 1) &lTR>…&l/TR> - новая строка таблицы; 2) &lTH>…&l/TH> - ячейка заголовка; 3) &lTD>…&l/TD> - обычная ячейка таблицы. 4) &lCAPTION>…&l/CAPTION> - заголовок таблицы В следующем примере показана простейшая таблица, состоящая из двух строк и двух столбцов, причем ячейки первой строки отведены на заголовки столбцов. 26 Пример простейшей пустой таблицы: &l!—Создаем объект таблицы--> &lTABLE> &l!—Создаем новую строку--> &lTR> &l!—Создаем две ячейки заголовков--> &lTH> &l/TH>&lTH> &l/TH> &l/TR> &l!—Создаем новую строку--> &lTR> &l!—Создаем две обычные ячейки--> &lTD> &l/TD>&lTD> &l/TD> &l/TR> &l!—Завершаем объект таблицы--> &l/TABLE> Но такие простые встречаются достаточно редко. В связи с этим, возникает необходимость в объединение строк или столбцов таблицы. Для объединения ячеек в тэгах &lTD> и &lTH> устанавливаются следующие параметры. 1) COLSPAN – число объединённых столбцов 2) ROWSPAN - число объединённых строк Для тэгов &lTD> и &lTH> также определены следующие атрибуты: a) ALIGN: горизонтальное выравнивание внутри ячейки 1) left – влево; 2) center – по центру; 3) right – вправо; b) VALIGN: вертикальное выравнивание внутри ячейки 1) top – вверх; 2) middle – по центру; 3) bottom – вниз; c) BACKGROUND – картинка на фон к ячейке d) BGCOLOR – цвет фона ячейки 27 Практические задания 1) Создайте новую страницу, назовите ее Таблицы.html 2) Создайте следующие таблицы: 1 2 3 1 2 3 4 5 Для каждой таблицы задать ширину и высоту по 200Px, границы вокруг таблицы по 2px, выравнивание внутри ячеек по центру. 3) Выполнить задания из файла Задания по теме Таблицы.doc 1 2 3 1 2 3 4 28 Разметка макета страницы с помощью таблицы В большинстве случаев данные на web – странице располагаются не по порядку сверху вниз, а размещаются блоками.
